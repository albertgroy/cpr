## 录入要求

- Bug、新需求、遗漏边界、契约缺口都应及时记入本表。
- 字段含义：
  - 来源：原始任务 ID 或反馈渠道。
  - 影响：阻塞验收 / 阻塞实现 / 影响体验 / 影响契约 等。
  - 处理决策：插入当前队列 / 进入 backlog / 进入 `docs/06-待优化/` / 触发需求重谈。
- 高优先级（P0/P1）插入当前滚动列表前部；低优先级进入 backlog。
- 若变更使契约失效，回到 `docs/04-会议记录/` 重开需求讨论再继续实现。
