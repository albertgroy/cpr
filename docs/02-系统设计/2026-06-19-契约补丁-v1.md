# 契约补丁 v1（2026-06-19）

## CommandNode

- `id` 必须匹配 `^[a-z][a-z0-9]*(\.[a-z0-9_{}-]+)*$`。
- `id == tokens.join(".")`，参数段在 `id` 中保留 `{name}`。
- `children` 为节点 ID 引用；加载分 parse → link 两阶段。
- `execute: true` 默认生成 `execute.command = tokens.join(" ")`；中间节点允许同时有 `execute` 与 `children`。
- 首批 `param.source` 仅允许 `sdk.candidates.java.identifiers` 与 `sdk.installed.java.versions`。

## CommandSession

- 解析采用最长前缀匹配，未匹配尾部写入 `unparsedTail`。
- `lastResult` 使用 N=5 环形缓冲，保留 `/fix` 所需上下文。
- Backspace 仅在输入区为空时弹出 token；Esc 在输入区非空时清空输入，输入区为空时弹出 token。

## CandidateProvider

- 接口为 `async def provide(...)`。
- 动态候选缓存 ttl=60s。
- `sdk install/use/default` 成功后必须失效 `sdk.installed.java.versions` 与 `sdk.candidates.java.identifiers`。
- 解析或执行失败返回空候选和 `error: {code, message}`。

## Executor

- `direct` 默认超时 30s；`bash-lc-source` 默认超时 120s。
- Linux 设置 `LC_ALL=C.UTF-8`，macOS 设置 `LC_ALL=en_US.UTF-8`；UTF-8 解码使用 `errors="replace"`。
- 默认 `cwd=$HOME`；环境白名单为 `HOME PATH LANG LC_ALL SDKMAN_DIR`。
- stdout/stderr 分流保留。
- `bash-lc-source` 优先 `$SDKMAN_DIR/bin/sdkman-init.sh`，否则 `~/.sdkman/bin/sdkman-init.sh`。
- `cancel()` 顺序为 SIGINT → 等待 2s → SIGTERM → 等待 2s → SIGKILL。

## SlashCommand

- Parser 文法为 `^/(\w+)(?:\s+(.*))?$`，name 小写，args 保留整行剩余内容。
- `/back` 复用 `session.popToken()` 语义。
- `/clear` 只清显示缓冲，保留 `lastCommand` 与 `lastResult`。
- 未知 slash 不进入历史。

## 跨契约

- 命令树加载失败时终端打印诊断并以退出码 2 终止，禁止半启动。
- locale 解析顺序为 `CPR_LOCALE` > 配置 > `LANG` > `en-US`。
- 日志路径为 `~/.cpr/logs/cpr.log`，warn 起记录，启动时打印路径。
