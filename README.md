# CPR

CPR 是当前工作代号：Command Prompt Runner / Command Path Recommender / Command Pilot Runtime。

项目目标是做一个面向命令行工具的交互式命令工作台：用户始终在一个主输入区里构造或执行真实命令，界面在同一窗口内根据当前命令层级展示说明、可选下一段、执行结果和 AI 辅助。
